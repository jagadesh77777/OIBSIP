# ATM INTERFACE

## Overview

This project is a console-based ATM Interface developed in Java as part of the Oasis Infobyte Java Development Internship. It demonstrates the use of Object-Oriented Programming (OOP) concepts by simulating basic ATM operations through a command-line interface.

## Features

- User Login using User ID and PIN
- Deposit Money
- Withdraw Money
- Transfer Money
- View Transaction History
- Input Validation for Invalid Transactions
- Menu-Driven Console Interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- VS Code

## How to Run

1. Clone or download the repository.
2. Open the project in VS Code or any Java IDE.
3. Compile the Java files:

javac *.java

4. Run the application:

java Main
